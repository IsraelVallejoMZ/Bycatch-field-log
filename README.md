# Bycatch Field Log

Offline-first PWA for field data capture during elasmobranch bycatch surveys. Built for port-based sampling in Thai demersal fisheries (Ranong, Phuket, Songkhla, Samut Sakorn).

## What it does

- Logs species, length, weight, sex, condition, gear type, and GPS position for each specimen, with optional photo evidence.
- Works fully offline — records are saved on-device and exported to CSV for downstream analysis (R, stock assessment, SDM workflows).
- Includes a map view that shows live tiles when a connection is available, and falls back to an offline coordinate plot when it isn't.
- Installable as a standalone app (PWA) on iOS and Android home screens.

## Status

Early-stage prototype, built to validate the data capture workflow before integrating it into a broader survey methodology.
